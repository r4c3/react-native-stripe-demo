Using the terminal:

1. Clone the GitHub Project
   ```
   $ git clone https://github.com/r4c3/react-native-stripe-demo.git
   ```

   ```
   $ cd client

   ```
3. Navigate to the frontend folder, download npm packages, and run
   ```
   $ cd server
   $ npm i
   $ npm run dev
   ```
